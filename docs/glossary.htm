Glossary

CALL
Invokes a procedure with optional parameters and result values. postgresql.org.
IS [ NOT ] DISTINCT FROM
A equality comparison operator. Handles operands with NULL values correctly.
FOUND
A special variable that indicates whether a query has found rows, or not.
DDL
Data Definition Language. A subset of SQL commands used to create and modify database objects.
